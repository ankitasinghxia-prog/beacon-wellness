# BEACON - Wellness Destination Website

![BEACON Website](https://beaconwellness.ct.ws)

A complete full-stack wellness website featuring Yoga, Trekking, and Cycling journeys.

## 🔗 Live Demo
[https://beaconwellness.ct.ws](https://beaconwellness.ct.ws)

## ✨ Features
- Scroll-triggered zoom animations with Three.js
- User authentication (Login/Signup)
- Contact form with email notifications
- Newsletter subscription
- Class booking system
- Admin dashboard
- Fully responsive design
- Glass-morphism UI

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Three.js
- **Backend:** PHP
- **Database:** MySQL
- **Deployment:** InfinityFree

## 📁 Project Structure

beacon-website/
├── index.html # Main website
├── styles.css # Styling
├── script.js # Frontend JavaScript
├── contact.php # Contact form handler
├── subscribe.php # Newsletter handler
├── register.php # User registration
├── login.php # User login
└── Assests/ # Images and videos


## 🚀 How to Run Locally
1. Install XAMPP
2. Clone this repository
3. Move to `htdocs/beacon/`
4. Import database from `database.sql`
5. Start Apache and MySQL
6. Visit `http://localhost/beacon/`
